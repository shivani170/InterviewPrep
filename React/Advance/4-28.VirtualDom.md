Virtual Dom:

- It is a lightweight copy of the actual dom.
- It is used to improve the performance of the application.
- It is used to reduce the number of re-renders.
- It is used to reduce the time it takes to update the dom.

Concept of reconciliation:

When state changes in react, react creates a new virtual dom tree. Then it compares the new virtual dom tree with the old virtual dom tree and finds the difference between them. Then it updates only the part of the dom that has changed.

Diffing Algo:

- It is used to find the difference between two virtual dom trees.
