Props Drilling:

- Passing down data through many layer of the component.
- Or can say from Parent to child to grand child in a unidirectional flow.

Context API:

- It is a way to pass data through many layer of the component without passing props to every component.
- It is a way to share data between components without prop drilling.
- It create a global space where these variables will automatically get stored and will be available to all the components
