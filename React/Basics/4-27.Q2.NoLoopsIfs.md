Q2. Why no loops/ifs directly

JSX looks like html but under the hood it acts like Js. It accepts expressions but statements.
JSX is syntactic sugar over React.createElement

which means JsX allows values and since loops and if statements are not expressions we use alternatives like ternary operators and array methods like map to render dynamic UI.

==========================================================  
❌ Wrong (statement inside JSX)
return (

  <div>
    {if (isLoggedIn) {
      <h1>Welcome</h1>
    }}
  </div>
);

👉 if is a statement, not allowed here
==========================================================  


✅ Correct (expression)
return (

  <div>
    {isLoggedIn ? <h1>Welcome</h1> : <h1>Login</h1>}
  </div>
);
