⚡ State (internal memory of the component)

- State = data managed inside the component

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <button onClick={() => setCount(count + 1)}>
      {count}
    </button>
  );
}
Key traits
🔄 Mutable (via setter)
🧠 Managed inside component
⚡ Changes trigger re-render
🎮 Controlled by the component itself

🧩 Props (inputs from the outside world)
Props = read-only data passed from parent → child

Key traits
🔒 Immutable (child cannot change them)
📦 Passed from parent
🎯 Used to customize components
🧭 Flow is top → down (one-way data flow)
